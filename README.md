# 📈 Stock Portfolio Tracker (Python)

A simple Python-based stock portfolio tracker that allows users to input stock names and quantities, then calculates total investment using predefined stock prices.  
This project demonstrates basic Python concepts like dictionaries, loops, file handling, and error handling in a real-world style simulation.

---

## ✨ Features

- 📊 Add multiple stocks with quantity input
- 💰 Predefined stock price dictionary
- 🧮 Automatic calculation of investment per stock
- 📦 Total portfolio value calculation
- ⚠️ Error handling for invalid input
- 💾 Saves portfolio summary to a `.txt` file

---

## 🛠️ Tech Stack

- Python  
- Dictionaries  
- Loops & Conditionals  
- File Handling  
- Exception Handling  

---

## ⚙️ How It Works

1. User enters stock symbol (e.g., AAPL, TSLA)
2. User enters quantity
3. Program calculates:
   investment = price × quantity
4. Displays:
- Individual stock investment
- Total portfolio value
5. Saves result in `portfolio.txt`

---

## ▶️ How to Run
python stock_tracker.py 

## 📁 Output Example
    TSLA -> 2 x 250 = 500
    AAPL -> 1 x 180 = 180
    Total Investment = 680
